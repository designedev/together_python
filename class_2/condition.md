# 조건문(IF)


> ## 특정 조건일 때 코드를 실행합니다.

- 핸드폰 벨소리가 울리면 -> 전화를 받는다. 를 코드로 표현하면?
```
if(핸드폰 벨소리가 울린다):
    전화를 받는다
```

- 미세먼지 농도가 심하다면 -> 마스크를 쓴다
```
if(미세먼지 농도가 심하다):
    마스크를 쓴다
```
- 비가 온다면 -> 우산을 쓴다
```
if(비가 온다):
    우산을 쓴다
```

> ## if 와 조건식을 지정하고 :(콜론) 을 붙인 뒤, 그 다음줄에 실행할 코드를 '들여쓰기로' 작성한다.

```
if 조건식:
(들여쓰기)코드

# 파이썬에서는 들여쓰기로 코드를 구분하기 때문
```

```
floor = 6
if floor == 6:
    print("좀 내립시다")

# 어떻게 될까요?
```

```
floor = 5
if floor == 6:
    print("좀 내립시다")

# 어떻게 될까요?
```

> ## 조건문의 실행 코드를 생략할 수 있습니다.
```
floor = 6
if floor == 6:
    print("좀 내립시다")

# 내릴 때 어떤 메세지를 출력할지 아직 미정이라면?

if floor == 6:
    pass
```
